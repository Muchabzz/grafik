# Grafik Orlen

Aplikacja do tworzenia grafiku pracy. Działa w przeglądarce i można ją dodać do ekranu głównego jak zwykłą aplikację.

## Jak uruchomić

1. Otwórz link do aplikacji w przeglądarce.
2. Po pierwszym uruchomieniu aplikacja zapisze się w pamięci urządzenia.
3. Każde urządzenie ma własne dane, więc grafik zapisany na telefonie nie pojawi się automatycznie na komputerze.

## Instalacja Na Windows

Najlepiej użyć Microsoft Edge albo Google Chrome.

### Microsoft Edge

1. Otwórz link do aplikacji.
2. Kliknij `...` w prawym górnym rogu.
3. Wybierz `Aplikacje`.
4. Kliknij `Zainstaluj tę witrynę jako aplikację`.
5. Po instalacji aplikację można przypiąć do paska zadań albo menu Start.

### Google Chrome

1. Otwórz link do aplikacji.
2. Kliknij ikonę instalacji po prawej stronie paska adresu, jeśli się pojawi.
3. Jeśli jej nie ma, kliknij `⋮` w prawym górnym rogu.
4. Wybierz `Zainstaluj stronę jako aplikację` albo `Utwórz skrót`.
5. Jeśli pojawi się opcja `Otwórz jako okno`, zaznacz ją.

## Instalacja Na Android

Najlepiej użyć Google Chrome.

1. Otwórz link do aplikacji w Chrome.
2. Kliknij `⋮` w prawym górnym rogu.
3. Wybierz `Dodaj do ekranu głównego` albo `Zainstaluj aplikację`.
4. Potwierdź dodanie.

Po tym aplikacja pojawi się na ekranie głównym telefonu.

## Instalacja Na iPhone

Na iPhonie użyj Safari.

1. Otwórz link do aplikacji w Safari.
2. Kliknij ikonę udostępniania.
3. Wybierz `Do ekranu początkowego`.
4. Kliknij `Dodaj`.

Ikona aplikacji pojawi się na ekranie początkowym.

## Działanie Offline

Po pierwszym wejściu aplikacja powinna działać także bez internetu. Jeśli aplikacja była niedawno aktualizowana, otwórz ją raz przy włączonym internecie, zamknij i uruchom ponownie.

## Gdy Nie Widać Opcji Instalacji

Sprawdź:

- czy link zaczyna się od `https://`,
- czy strona została odświeżona po ostatniej aktualizacji,
- czy otwierasz aplikację w Chrome, Edge albo Safari,
- czy aplikacja nie jest już zainstalowana.

Na komputerze zawsze można też dodać zwykły skrót do strony, ale najlepszy efekt daje opcja instalacji aplikacji.

## Eksport Do Excela

W aplikacji kliknij `Pobierz Excel`. Plik zapisze się na urządzeniu tak jak zwykły pobrany plik z przeglądarki.
